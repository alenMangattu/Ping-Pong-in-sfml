# Pong Game
Implementazione facile del gioco "Pong" in C++ usando la libreria SFML.

```
g++ src/main.cpp -o PongGame -lsfml-graphics -lsfml-window -lsfml-system
```

## Tasti
- Giocatore 1: W (up), S (down)
- Giocatore 2: Freccia Su (up), Freccia Giu (down)
